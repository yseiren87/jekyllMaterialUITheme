# List

## List

### Ordered List

<pre>
1. a
2. b
3. c
</pre>

<pre>
1. a
2. b
    1. ba
    2. bb
3. c
</pre>

### Unordered List

<pre>
- a
- b
- c
</pre>

<pre>
* a
* b
* c
</pre>

<pre>
- a
- b
    - ba
    - bb
- c
</pre>

### Adding Elements in Lists

#### Paragraphs
<pre>
* a
* b

    sdfasdf
    
* c

</pre>

####  Blockquotes

<pre>
* a
* b
    
    > asdf
    
* c
</pre>

#### Code Blocks
<pre>
* a
* b

        <html>
            <head>
                <title>Test</title>
            </head>
        </html>

* c
</pre>

#### Images

<pre>
* a
* b
    
    ![Tux](/assets/images/tux.png)
    
* c
</pre>

### Definition Lists
<pre>
First Term
: This is the definition of the first term.

Second Term
: This is one definition of the second term.
: This is another definition of the second term.
</pre>
